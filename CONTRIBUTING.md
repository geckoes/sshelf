# Contributing to sshelf

Thank you for your interest in contributing! 
Every contribution is welcome, from bug reports to new features.

## How to report a bug
Open an issue, describe it and attach log

## How to suggest a feature
Open an issue labeled _enhancement_

## How to submit a Pull Request
Fork the repository , create your branch, submit your PR
Please use descriptive branch names like `fix/ssh-parse-error` or `feat/search-host`.
