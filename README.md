# sshelf

A lightweight TUI to manage SSH config easily.

## Features

You can:

- **see** hosts configured in your ssh config
- **add** a new host (Host, HostName, User, port, IdentityFile)
- **modify** an existing host
- **delete** a host
- **look for** a host by its name
