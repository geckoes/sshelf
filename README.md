# sshelf

A lightweight TUI to manage SSH config easily.

## Features
You can:
- __see__ hosts configured in your ssh config
- __add__ a new host (Host, HostName, User, port, IdentityFile)
- __modify__ an existing host
- __delete__ a host
- __look for__ a host by its name
